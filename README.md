# 2048 Game

Console-based 2048 puzzle game built in C++.

## Tech Stack

C++

## Features

* Classic 4×4 2048 gameplay
* Move tiles using arrow keys
* Automatic tile generation (2 or 4)
* Tile merging and score tracking
* Win condition at 2048
* Game over detection
* Quit anytime by pressing `Q`

## How to Run

```bash
g++ 2048game.cpp -o 2048game
.\2048game
```
Or simply run in any code editor (Vscode, Dev c++)

## Controls

* ↑ : Move Up
* ↓ : Move Down
* ← : Move Left
* → : Move Right
* Q : Quit the game

## Objective

Combine tiles with the same numbers to create the `2048` tile while achieving the highest possible score.
